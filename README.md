# Env variables

```sh
# Set env variable
$env:APP_BACKEND_URL="http://localhost:3001/api/0.01/"

# Delete env variable
Remove-Item Env:\APP_BACKEND_URL

# See current env variables in windows:
Get-ChildItem env:

# See current env variables in Linux:
printenv
```
